# Lawyance Intro

Independent Astro + Vue 3 introduction page for Lawyance.

This site is intentionally separate from the production Lawyance application and from the Lawyance Design System source folder. The design references only the Lawyance brand-logo component direction.

## Commands

```bash
pnpm install
pnpm dev
pnpm build
```

Build output is written to `dist/` for Cloudflare Pages or any static host.
